Using the Circles Vector Graphics Editor

Drawing a single circle

Left-click with the mouse on the drawing surface.

Drawing a line

Left-click with the mouse on the drawing surface and drag the mouse while holding down the left mouse button.

Resizing the canvas

Scroll the mouse wheel up or down to resize the drawing surface.

Moving the application window

Left-click and drag on the transparent frame surrounding the drawing surface.
